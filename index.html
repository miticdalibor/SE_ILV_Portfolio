Dieses Portfolio dient als Dokumentation von Frunch Infinity 2.0. Frunch Infinity ist eine marktführende Softwarelösung, welche in allen Geschäftsbereichen unterschiedlicher Industrien eingesetzt werden kann. Damit können Benutzer mit ihren Daten unterschiedliche konventionellen Modellen ohne Programmierkenntnisse anlernen und dadurch schnelle Einblicke in die Qualität und Verwendbarkeit ihrer Daten von ML-Lösungen erhalten. Mit der Applikation können die User auch individuelle Problemstellungen lösen, indem die notwendigen Modellstrukturen sowie Vorverarbeitungsschritte der Daten automatisch erstellt werden und somit das bekannte Free Lunch Theorem aufgehoben wird. Dieses Portfolio beschreibt das fiktive Endprodukt von Frunch Infinity, dessen Teilimplementierung im Lab von Softwareentwicklung 2 (DSIA SS 2022) im Zuge einer Projektarbeit erfolgt. Um das Portfolio von der Projektarbeit zu trennen, wurde der Titel dieses Portfolios auf Frunch Infinity 2.0 definiert. # Inhaltsverzeichnis
