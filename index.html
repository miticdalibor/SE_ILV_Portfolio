Dieses Portfolio dient als Dokumentation von Frunch Infinity 2.0. Frunch Infinity ist eine fiktive Softwarelösung, welche in allen Geschäftsbereichen unterschiedlicher Industrien eingesetzt werden kann. Damit können Benutzer mit ihren Daten unterschiedliche ML-Modellen ohne Programmierkenntnisse anlernen, validieren und testen lassen. Dadurch können die User schnelle Einblicke in die Qualität und Verwendbarkeit ihrer Daten von ML-Lösungen erhalten. Mit der Applikation können die User auch individuelle Problemstellungen lösen, indem die notwendigen Modellstrukturen sowie Vorverarbeitungsschritte der Daten automatisch erstellt werden und somit das bekannte Free Lunch Theorem aufgehoben wird. Dieses Portfolio beschreibt das fiktive Endprodukt von Frunch Infinity, dessen Teilimplementierung im Lab von Softwareentwicklung 2 (DSIA SS 2022) im Zuge einer Projektarbeit erfolgt. Das Repository zum Projekt kann über folgenden Link geöffnet werden. Um das Portfolio von der Projektarbeit zu trennen, wurde der Titel dieses Portfolios auf Frunch Infinity 2.0 definiert.

Lessons Learned: Die Schwierigkeit von Frunch Infinity 2.0 liegt in der Komplexität des Systems. Dabei muss das System flexibel und skalierbar sein und zugleich alle Edge Cases im Preprocessing abgreifen. Dies ist schwierig am Anfang, ohne die zu verarbeitenden Datensätze zu kennen. Deshalb kam die Lösung den Preprocessing Teil vom User überwachen zu lassen, damit er entsprechend auch eingreifen kann. Ich glaube, am Ende hat man selber keinen Überblick mehr, was das System bewältigen kann. Deshalb ist hier die Dokumentation des Codes ein wichtiger Teil und die Architektur muss sehr überdacht gewählt werden. Durch die Anwendung von den Tools aus der ILV, hat man aber einen guten Werksatz, wo man für so ein komplexes Projekt ansetzen kann und sich selber mit dem System weiterentwickeln kann.
